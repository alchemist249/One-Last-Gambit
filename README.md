# One last Gambit

A turn-based battle game in **Ren'Py** with stylized combat, cinematic UI, and dynamic enemy responses.

## Features
- Turn-Based Combat: Attack, Defend, Counter, Taunt.
- Dynamic Enemy AI reacting to your choices.
- Cinematic UI with smooth animations.
- Branching outcomes affecting story and battle.
- Post-Boss Upgrades to enhance strategy.

## Installation
1. Install [Ren'Py](https://www.renpy.org/latest.html)
2. Clone repo:
3. Navigate
4. Launch .py with Ren'Py Launcher.

## Gameplay
Pokemon-esque turn-based combat and roguelike features
- **Attack:** Deal damage.
- **Defend:** Reduce damage.
- **Counter:** Retaliate after block.
- **Gambit:** Gambles to win or lose.
